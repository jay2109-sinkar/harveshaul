# HarvestHaul

Lightweight Flask prototype implementing a multi-agent food rescue matchmaker.

Quick start

1. Create a virtualenv and install deps:

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

2. Run the app:

```bash
python app.py
```

3. Open http://127.0.0.1:5000/

Data is stored under the `data/` directory as JSON files.
